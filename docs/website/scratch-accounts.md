# Scratch 账户 和 该编辑器

虽然您无法在 TurboWarp（任何除了 scratch.mit.edu 之外的任何要求您输入 Scratch 密码的网站都是在搞诈骗——这绝对没有例外）上登录您的 Scratch 账户，但您仍然可以在不分享它们的情况下运行作品，也可以使用自己的用户名将作品上传至 Scratch 。

**在不分享的情况下使用 TurboWarp 来运行 Scratch 编写的作品**: 在 Scratch 编辑器中，选择“文件>保存”来把作品保存到您计算机上；然后在 TurboWarp 中，选择“文件>从计算机加载”打开该文件。

**将使用 TurboWarp 制作的作品上传至 Scratch**: 在 TurboWarp 的编辑器中，选择“文件>保存”将作品保存到您计算机上的文件中；在 Scratch 的编辑器中，选择“文件>从计算机加载”来打开该文件。

**更改 TurboWarp 中的用户名**: 通过“编辑>更改用户名”来更改。

即使你在另一个标签页中已登录到 Scratch 网站，TurboWarp 也无法访问它，因为浏览器会阻止网站获取其他网站的登录信息（这是保障网上银行安全的重要措施）。我们如果向您索要 Scratch 的登录密码，会带来无法解决的安全隐患，并且很可能导致我们被 Scratch 封禁，这样你就再也用不了这么好的运行内核了。

根据您所要完成的任务，您可能会发现使用 [TurboWarp 包装器](https://packager.turbowarp.org/) 能够将您的作品上传至通用的 HTML 或游戏托管网站方。
